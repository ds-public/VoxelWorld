Simple Sprite Packer
 Version 2015/07/15_0 Programmed by Daigo Sakamoto

===========================================================
◆使い方

1)

メニューの

 Tools -> Simple Sprite Packer

で、起動してください。


2)
アトラスタイプのスプライトを書き出したいフォルダを選択し、
[Atlas Path]ボタンを押してください。

適当なファイル名を入力してください。

※
既に存在するアトラスタイプのスプライトを編集したい場合は、
そのアトラスタイプのスプライトを選択して、[Atlas Path]ボタンを押してください。


3)

アトラスタイプのスプライトに加えたい、
テクスチャ（スプライト）を選択（複数可能）して、
[Create]または[Update]ボタンをおしてください。

※
Padding の値で、要素画像間の隙間（ピクセル数）を設定することができます。

要素画像リストの表示：
Update …要素画像が更新が更新されます
Add    …要素画像が追加されます
Delete(赤) …既にアトラスタイプのスプライトに含まれている要素画像が削除されます。


4)

Output Divided Sprite にチェックを入れ、
[Output Path]を設定し、[Execute]ボタンを押すことにより、
アトラスタイプのスプライトに含まれる要素画像を、
個々のシングルタイプのスプライトとして書き出すことができます。
