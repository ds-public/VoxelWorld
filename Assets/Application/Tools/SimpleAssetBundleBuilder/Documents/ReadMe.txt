1)
　単体のファイルをアセットバンドル化する場合の記述方法(ファイルが無い場合は同名フォルダをアセットバンドル化対象とみなす)
　　/[Folder Name]/[File Name]

2)
　複数のファイルまたはフォルダをアセットバンドル化する場合の記述方法

　　/[Folder Name]/*

3)
　単体のフォルダをアセットバンドル化する場合の記述方法(フォルダ限定)

　　/[Folder Name]/

4)
　複数のフォルダをアセットバンドル化する場合の記述方法(フォルダ限定)

　　/[Folder Name]/*/

以下のような記述方法も可能
　/[Folder Name]/*/*
　/[Folder Name]/*/*/
　/[Folder Name A]/*/[Folder Name B]/*
　/[Folder Name A]/*/[Folder Name B]/*/

==============================================================

[コメントアウト]
　先頭が # の場合は、その行は無効化されます。

[アセットパンドル化を行わない]
　先頭が ~ の場合は、
　1)2)のケースのみ有効で、対象のファイルをアセットバンドル化を行わずそのまま管理対象に加え複製する。
　3)4)のケースでは無視される。
