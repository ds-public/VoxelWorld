◆Simple PNG Integrator の使い方

　Version 2021/03/01

-------------------------------------------------------------------------------
●概要

　PNG 以外の無圧縮・可逆圧縮の画像フォーマットを PNG に変換し、
　参照を全て貼り直します。


-------------------------------------------------------------------------------
●使い方

　1)
　　変換を施したいファイル群が含まれたルートフォルダを選択し、
　　[Root AssetPath] を押します。

　2)
　　変換対象となるファイル一覧が表示され、
　　下部に [Modify] ボタンが表示されるので押します。


　※
　　Delete files with 0 references. にチェックを入れて [Modify] ボタンを押すと、
　　PNG へ変換後に、変換前のファイルを削除します。


-------------------------------------------------------------------------------
●注意点

　1)
　　[Root AssetPath] フォルダ外から、
　　変換対象ファイルへの参照を持っている場合、
　　変換対象ファイルが削除されると、Missing を起こします。
　　※そのような構造は、元々問題なのですが。


　2)
　　変換後の PNG のサイズは、Inspectr の Max Size の影響を受けます。

　　例えば、
　　オリジナルのテクスチャのサイズが 2048 x 2048、
　　Max Size が 1024 に設定されていると、
　　変換後の PNG のサイズは、1024 1024 になります。
　　※オリジナルのテクスチャのサイズは取得できないため。
