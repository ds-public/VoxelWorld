このツールは、Unityプロジェクト内に置かれたExcelファイル(拡張子 *.xls 推奨)から、
CSV形式のテキストファイルを生成するためのツールです。

CSV形式のテキストファイルは、Excelファイルを保存(新規・上書き)した際に、
指定した書式に沿って、自動的に生成・保存されます。

================================================================
■Excelファイルの書式

1)
@Export という名前のシートを作成してください。
そして、そのシートの A1 セルに、
/Assets から始まる、CSV形式のテキストファイルの出力フォルダのパスを記述してください。

例
　@Export シートの A1 セル

　/Assets/Application/AssetBundle/Data/MasterData

2)
@Export 以外のシートが、1)で指定したフォルダに、CSV形式のテキストファイルとして出力されるようになります。
CSV形式のテキストファイルのファイル名は、シート名.txt になります。
CSV形式のテキストファイル内の各カラムは、タブで区切られます。

※シート名の最初を # とする事で、そのシートを出力しないようにする事ができます。

例
　シート名 EnemyUnit というシートが存在する場合、以下のように出力される。

　/Assets/Application/AssetBundle/Data/MasterData/EnemyUnit.txt


================================================================
■重要

本ツールの使用には、NPOI 2.4.1 以上が必要になります。
本ツールを使用するUnityプロジェクトに NPOI 2.4.1 以上のモジュール(DLL)を導入してください。
https://www.nuget.org/packages/NPOI/
